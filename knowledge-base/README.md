# 📚 知識庫
## 目錄用途
集中存儲所有靜態知識、概念、架構、工具使用說明等。
## 子目錄說明
| 子目錄 | 用途 |
|-------|------|
| `core-concepts/` | 核心概念解釋，術語定義 |
| `architecture/` | 系統架構設計、技術選型、架構模式 |
| `tools/` | 各種工具的使用說明、最佳實踐、常見問題 |
| `integrations/` | 第三方系統、服務、API 的集成文檔 |
## 格式要求
- 所有文檔使用 Markdown 格式
- 包含清晰的標題和層級結構
- 示例代碼使用代碼塊
- 複雜概念附上示意圖
