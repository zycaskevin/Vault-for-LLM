# Vault-for-LLM v0.6.42

This is a documentation polish release for the agent installer and Supabase setup path.

- Made `docs/agent_install.md` easier for agents and humans to follow by starting with five essential questions, then introducing optional features only when needed.

No runtime behavior changed in this release.
