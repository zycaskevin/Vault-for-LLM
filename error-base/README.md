# ❌ 錯誤庫
## 目錄用途
存儲所有曾經發生過的錯誤、根本原因、解決方案、預防措施，確保錯誤不會重複發生。
## 子目錄說明
| 子目錄 | 用途 |
|-------|------|
| `error-catalog/` | 錯誤目錄，按類型分類的所有已知錯誤 |
| `root-cause/` | 根本原因分析，深入分析錯誤發生的根本原因 |
| `fix-solutions/` | 解決方案，詳細的修復步驟 |
| `prevention/` | 預防措施，如何避免同樣錯誤再次發生 |
## 格式要求
- 所有錯誤條目必須遵循 [錯誤條目格式](../../CONTRIBUTING.md#錯誤條目)
- 包含完整的錯誤信息、重現步驟、解決方案、預防措施
- 記錄錯誤發生次數和最後發生時間
