# Vault-for-LLM v0.7.17

v0.7.17 deepens the automatic memory loop.

The focus is not more commands. It is better judgment inside the existing
pipeline:

- Session capture now annotates candidates with extraction score, novelty
  score, recommended action, and merge target.
- Search ranking now prefers currently valid facts while keeping past facts
  auditable.
- Reflection can propose consolidation candidates for similar active memories
  without rewriting or deleting the originals.

Recommended upgrade:

```bash
pip install --upgrade "vault-for-llm[mcp]==0.7.17"
```

No manual migration is required.
