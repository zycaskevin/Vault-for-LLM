# 機器學習概念定義
## 核心定義
機器學習（Machine Learning）是人工智慧的一個分支，它使計算機系統能夠從數據中學習並改進性能，而無需明確編程。
## 關鍵特徵
1. **數據驅動**：依賴大量數據進行訓練
2. **自動改進**：隨著更多數據的輸入而提高準確性
3. **模式識別**：自動發現數據中的模式和關係
4. **預測能力**：基於歷史數據進行未來預測
## 主要類型
- **監督式學習**：使用標記數據進行訓練
- **非監督式學習**：從未標記數據中發現模式
- **強化學習**：通過獎懲機制學習最佳策略
## 相關概念
- 深度學習
- 神經網絡
- 特徵工程
- 過擬合與欠擬合
