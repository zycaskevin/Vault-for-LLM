# Vault-for-LLM v0.7.14

v0.7.14 is a maintenance release focused on keeping the automation system easier to review and extend.

The automation policy defaults and parsing helpers now live in `vault/automation_policy.py`, while the existing `vault.automation` import surface remains compatible. This keeps user-facing behavior stable and reduces the size of the main automation workflow module.

Recommended upgrade:

```bash
pip install --upgrade "vault-for-llm[mcp]==0.7.14"
```

No database migration or configuration change is required.
