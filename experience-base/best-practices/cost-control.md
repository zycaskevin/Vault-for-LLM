# 成本控制最佳實踐

## 概述
AI 應用的 API 成本可能迅速失控，需要嚴格的成本控制機制。

## 核心策略

### 1. Token 截斷
- 4,000 Tokens 截斷
- 永不發送完整歷史

### 2. 記憶摘要
- 每夜記憶摘要
- 壓縮日常對話為高密度標籤

### 3. 預扣除機制
- API 調用前扣除點數
- 失敗自動退款

### 4. 隱藏上限
- 每用戶 300 條消息/天限制
- 超過後降級到簡單模式

### 5. AUTO 模式
- 意圖分類器自動路由
- 根據任務複雜度選擇模型

## 價值
將不可控成本轉化為可預測的定價模式。
