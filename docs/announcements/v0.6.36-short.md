# Vault-for-LLM v0.6.36

This release productizes Supabase remote readers for agents and automations.

## Highlights

- Added `vault remote smoke` for one-command Supabase remote reader verification.
- Added `vault setup-agent --remote-reader shell|n8n|coze|all`.
- The installer can now generate shell, n8n, and Coze remote reader templates.
- Remote readers use safe summaries through `vault_search_readable`; service role keys stay on trusted sync hosts.

Use this when Hermes, Codex, Claude Code, OpenClaw, n8n, Coze, or another host needs to read the same reviewed memory view without sharing a local `vault.db`.
