# 💡 經驗庫
## 目錄用途
存儲所有最佳實踐、案例研究、經驗教訓、成功案例，是過去經驗的沉澱，用於對抗系統熵增。
## 子目錄說明
| 子目錄 | 用途 |
|-------|------|
| `best-practices/` | 各領域的最佳實踐，經過驗證的有效做法 |
| `case-studies/` | 完整的案例研究，包含背景、問題、解決方案、結果 |
| `lessons-learned/` | 經驗教訓，必須遵循 [傷疤格式](../../CONTRIBUTING.md#經驗教訓傷疤格式) |
| `success-stories/` | 成功案例，值得複製和推廣的成功經驗 |
## 格式要求
- 經驗教訓必須嚴格遵循傷疤格式
- 案例研究包含完整的背景、過程、結果、可複製性說明
- 最佳實踐包含適用場景、實施步驟、預期效果
