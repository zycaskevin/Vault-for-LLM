# Ollama Cloud限流問題

Ollama Cloud API 限流導致請求失敗，需注意並行模型數、GPU時間計算及限流重置週期。

## 關鍵要點
- 同時最多3個雲端模型並行，超限請求會被拒絕或排隊
- 用量以GPU時間計算，長prompt與大模型會增加用量
- 超過限流會導致timeout、空回應、429錯誤或400錯誤
