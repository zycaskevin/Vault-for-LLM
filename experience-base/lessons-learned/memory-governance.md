## 五席位治理機制

1. **立法席位**：檢查記憶是否有資格寫入（能改變行為）
2. **修憲席位**：保留歷史版本，不可靜默覆蓋
3. **司法席位**：解決衝突記憶，按場景優先裁決
4. **審計席位**：遺忘責任追蹤，記錄審計日誌
5. **交接席位**：生成交接狀態包，包含未完成承諾
