# Vault-for-LLM v0.6.100

This release adds the shortest human review surface for memory automation.

`vault automation review-summary` turns the brief, inbox, and latest report into
a few approval cards. Each card tells you what needs attention, why it matters,
what action is recommended, and what the safe next step is.

Why it matters:

- humans can review the 5% that actually needs judgment
- agents can hand off automation state without raw candidate content
- dashboards can show one compact approval queue
- the command is read-only: no promotion, no archive, no deletion, no candidate
  writes

Upgrade:

```bash
pip install -U "vault-for-llm[mcp]==0.6.100"
vault automation review-summary --write-summary
```
