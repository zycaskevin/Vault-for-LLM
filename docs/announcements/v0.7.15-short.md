# Vault-for-LLM v0.7.15

v0.7.15 is a Task Ledger continuity release.

Task Ledger gives agents a short-lived working set for active tasks, blockers,
next actions, evidence links, and handoff notes. It is intentionally separate
from L0-L3 long-term memory: it helps the next agent resume the work without
turning every in-progress todo into permanent knowledge.

This release surfaces Task Ledger across the product:

- CLI task lifecycle commands for active work.
- MCP Task Ledger tools in review, maintenance, and full profiles.
- A read-only GUI Active Tasks panel.
- Automation cycle handoffs that include compact task snapshots.

Recommended upgrade:

```bash
pip install --upgrade "vault-for-llm[mcp]==0.7.15"
```

No manual migration or configuration change is required. Vault runs its normal
schema migration when it opens a project database.
