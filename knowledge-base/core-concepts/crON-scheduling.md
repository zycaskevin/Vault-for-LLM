# Cron 定時任務

## 概述
Cron 是 Linux 系統的定時任務工具。

## 語法
```
┌───────────── 分鐘 (0 - 59)
│ ┌───────────── 小時 (0 - 23)
│ │ ┌───────────── 日期 (1 - 31)
│ │ │ ┌───────────── 月份 (1 - 12)
│ │ │ │ ┌───────────── 星期 (0 - 6)
│ │ │ │ │
* * * * *
```

## 示例

### 每小時執行
```
0 * * * * /path/to/script.sh
```

### 每天凌晨 3 點
```
0 3 * * * /path/to/script.sh
```

### 每週一 9 點
```
0 9 * * 1 /path/to/script.sh
```
