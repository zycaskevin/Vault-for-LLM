# Vault-for-LLM v0.7.6

v0.7.6 makes the automatic memory loop easier for humans to review.

What changed:

- Generated cron, LaunchAgent, and n8n schedules now also run:
  - `vault automation review-summary --write-summary`
- Scheduled runs now produce:
  - `reports/automation/review-summary-latest.json`
  - `reports/automation/review-summary-latest.md`
- Review-summary Markdown now reads like a compact decision-card deck:
  - suggested decision
  - why it matters
  - safe next step

Safety boundary:

- Review-summary is read-only.
- It does not show raw candidate content.
- It does not promote, archive, delete, or widen policy.
