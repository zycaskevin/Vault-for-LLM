# Vault-for-LLM v0.6.38

This is a privacy hotfix release for the multi-agent installer docs and examples.

## Highlights

- Replaced private/local agent names in public examples with generic role IDs such as `profile-agent`, `work-agent`, `product-agent`, and `remote-agent`.
- Sanitized README variants, agent install docs, CLI reference, manifest examples, changelog text, benchmark notes, release-note drafts, and tests.
- Kept the v0.6.37 multi-agent roster and live validation features intact.
- Added a repository privacy review pass for private names, local user paths, emails, tokens, and credential-like patterns.

Use this release instead of v0.6.37 for public installs and references.
