# Vault-for-LLM v0.7.0rc1

v0.7.0rc1 is the first release candidate for the Agent Knowledge Platform foundation.

This RC does not add a new hosted dependency or background automation trigger. It marks the point where the 0.6 hardening line becomes a platform boundary: local-first memory, MCP access, agent installation, bounded reads, automation briefs, update rollout health, and governance metadata are expected to work together as one product surface.

The release gate now accepts PyPI-compatible release-candidate versions such as `v0.7.0rc1`, so GitHub Releases, PyPI, `pyproject.toml`, `vault.__version__`, and `CHANGELOG.md` can stay in parity during RC testing.

Use this RC to test clean installs and Agent startup flows before v0.7.0 stable.
