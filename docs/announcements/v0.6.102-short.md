# Vault-for-LLM v0.6.102

This release adds a compact health panel for the automation learning loop.

`vault automation learning-health` shows whether feedback learning is still in
cold start, healthy, worth watching, or needs review. It summarizes accepted,
rejected, and deferred outcomes without exposing raw feedback reasons.

Why it matters:

- humans get one short automation-learning status
- agents can start from a dashboard-safe health signal
- repeated review feedback becomes easier to audit
- the command is read-only: no promotion, archive, deletion, or policy mutation

Upgrade:

```bash
pip install -U "vault-for-llm[mcp]==0.6.102"
vault automation learning-health --write-health
```
