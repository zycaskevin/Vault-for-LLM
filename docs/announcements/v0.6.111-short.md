# Vault-for-LLM v0.6.111

This release continues the MCP modularization work.

## Highlights

- Moved Supabase remote MCP reader logic into `vault.mcp_remote`.
- Kept existing `vault.mcp` helper imports compatible.
- Reduced the `vault/mcp.py` size baseline from 3132 lines to 2248 lines.

## Why It Matters

Remote search, map, doctor, and bounded read are security-sensitive paths. Giving them their own module makes future review easier without changing public MCP tool names or payloads.

## Upgrade

```bash
pip install -U "vault-for-llm[mcp]==0.6.111"
```

No configuration migration is required.
