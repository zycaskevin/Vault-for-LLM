# Vault-for-LLM v0.6.121

This release continues the module-size cleanup by moving semantic CLI command handling into `vault.cli_semantic`.

## What Changed

- Split `vault semantic` handlers and helper functions out of the large `vault.cli` module.
- Preserved the existing `vault.cli` import paths for downstream compatibility.
- Updated the module-size baseline for `vault/cli.py`.

## Compatibility

No CLI behavior changes are intended. Existing `vault semantic rebuild`, `warm`, `smoke`, cache, startup, and daemon commands should behave the same as v0.6.120.

## Verification

The release was checked with focused CLI/semantic tests, module-size gates, release parity checks, README command smoke tests, public PR gates, and a clean PyPI install smoke.
