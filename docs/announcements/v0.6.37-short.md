# Vault-for-LLM v0.6.37

This release adds deployment-ready multi-agent setup files and live validation checklists.

## Highlights

- Added `vault setup-agent --agent-roster`.
- Added `vault setup-agent --validation-pack remote|n8n|coze|all`.
- The installer can generate `agent-roster.json`, `AGENT_ACCESS_MATRIX.md`, per-agent env examples, and setup commands.
- The installer can generate `validate-remote-reader.sh`, `VALIDATE-n8n.md`, `VALIDATE-coze.md`, and `README-live-validation.md`.

Use this when a team wants Nancy, Mori, Aiko, Coco, n8n, Codex, or other agents to share reviewed memory while keeping private persona and profile memory separate.
