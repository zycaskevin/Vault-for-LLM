# Vault-for-LLM v0.6.120

This release continues the module-size and maintainability pass.

## What Changed

- Split search reranking helpers into `vault.search_rerank`.
- Kept the existing `vault.search` import path working.
- Reduced the size of `vault/search.py` so retrieval and ranking can evolve more
  independently.

## Safety Notes

- Search behavior is unchanged.
- Rerank score fields are unchanged.
- Cross-encoder reranking remains optional and local to the configured runtime.

## Upgrade

```bash
pip install -U "vault-for-llm[mcp]==0.6.120"
```
