# Vault-for-LLM v0.6.122

This release continues MCP module cleanup by moving local Document Map and bounded-read helpers into `vault.mcp_read`.

## What Changed

- Split local MCP `vault_map_show` and `vault_read_range` payload logic out of the large `vault.mcp` module.
- Preserved existing `vault.mcp` import paths for compatibility.
- Updated the module-size baseline for `vault/mcp.py`.

## Compatibility

No MCP tool behavior changes are intended. Local map/read outputs, citations, access filtering, and error guidance should match v0.6.121.

## Verification

The release was checked with focused MCP/read tests, module-size gates, release parity checks, README command smoke tests, public PR gates, full test suite, and a clean PyPI install smoke.
