## Vault-for-LLM v0.7.13

v0.7.13 is a maintenance release that pays down module size after the v0.7.12
review-hardening work.

- MCP search payload shaping now lives in `vault/mcp_search.py`.
- CLI temporal search option wiring now lives in `vault/cli_search.py`.
- Graph-expanded recall now lives in `vault/search_graph.py` and keeps the same
  read-policy guard added in v0.7.11.
- The module-size baseline is tightened for `cli.py`, `mcp.py`, and `search.py`.

No database migration or behavior change is required. This release keeps the
public CLI/MCP behavior stable while making the code easier to keep safe.
