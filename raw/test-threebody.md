# 第一章 瘋狂年代

葉文潔在清華大學的父親葉哲泰被紅衛兵批鬥致死。這段經歷在她心中種下了對人類文明深刻的懷疑。她在插隊期間，目睹了大興安嶺原始森林的砍伐，感受到人類對自然的破壞毫無節制。

葉文潔後來被調入紅岸基地，一個位於大興安嶺深處的軍事設施。她的物理學背景讓她成為基地中不可或缺的人才。紅岸基地的真正目的，是向太空發射信號，尋找外星文明。

# 第二章 太陽放大器

葉文潔發現太陽可以作為信號放大器。她利用紅岸基地的發射設備，向太陽發射了一束電磁波。太陽的等離子體將信號放大了數億倍，向宇宙深處廣播。這是人類第一次向宇宙發出有意義的信號。

八年後，來自半人馬座α星的三體文明收到了地球的信號。三體世界有三顆太陽，文明在混沌紀元中反覆毀滅和重建。三體人渴望一個穩定的恆星系統，而地球恰好就是這樣的星球。

# 第三章 三體遊戲

汪淼被邀請參加一個叫「三體」的虛擬現實遊戲。在遊戲中，他體驗了三體世界的殘酷——文明的每一次進步都被混亂的太陽運動摧毀。遊戲其實是三體向地球人展示其文明處境的媒介。

透過遊戲，汪淼逐漸理解了三體文明的本質：他們的科學發達但社會冷酷，生存是唯一的法則。三體人已經向地球派出了艦隊，預計四百年後到達。

# 第四章 面壁者

聯合國啟動了「面壁計劃」，選出四位面壁者。面壁者可以在大腦中制定戰略，不需要向任何人解釋。三體人的智子可以監視一切，但無法讀取人類的思想。這是人類唯一的優勢。

羅輯是四位面壁者中最特殊的一個。他不是軍事家，不是科學家，甚至不願意面壁。但他的「宇宙社會學」理論——黑暗森林法則——後來證明是人類最強大的武器。

# 第五章 黑暗森林

黑暗森林法則指出：宇宙中的文明彼此猜疑，任何暴露自己位置的文明都會被消滅。因此，保持沉默是生存的唯一策略。羅輯利用這個法則，威脅要向宇宙廣播三體星系的坐標，迫使三體艦隊轉向。

這個威脅建立了脆弱的威懾平衡。羅輯成為「執劍人」，手握兩個文明的命運。但威懾終究依賴個人的意志，當程心接替成為新的執劍人時，三體判斷她不敢按下按鈕，攻勢立刻重啟。
