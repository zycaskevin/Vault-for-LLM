# WSL2 Chrome CDP 桥接

解決WSL2無法直連Windows Chrome CDP的問題，採用三層橋接方案。

## 關鍵要點
- Chrome設置`--remote-debugging-port=9222`並綁定`127.0.0.1`
- Windows使用Node.js轉發`0.0.0.0:9223 → 127.0.0.1:9222`
- WSL使用Python aiohttp代理`0.0.0.0:3456 → <gateway>:9223`
