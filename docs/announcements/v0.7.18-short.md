# Vault-for-LLM v0.7.18

v0.7.18 makes the Skill Registry usable from MCP without turning agents into
silent runtime installers.

## Highlights

- Added `vault_skill_push` for explicit, privacy-gated registry writes.
- Added Skill sync status and manifest tools so external workers can sync by
  metadata/hash first, then opt into bounded content export when trusted.
- Added `vault_skill_mark_synced` so sync workers can record successful handoff.
- Kept Skill write tools out of `core`; `review` can inspect, `maintenance` can
  write, and no MCP Skill tool overwrites runtime Skill files.

## Install

```bash
pip install --upgrade "vault-for-llm[mcp]==0.7.18"
```
