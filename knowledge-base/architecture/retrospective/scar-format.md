# 傷疤格式 (Scar Format)

## 概述
傷疤格式是一種結構化的記憶記錄格式，用於將錯誤和經驗轉化為可執行的行為改變。

## 格式定義

```markdown
[Situation] 什麼情況？
[Wrong] 我做錯了什麼？
[Correct] 應該怎麼做？
[Behavior Change] 這會如何改變我未來的行為？
[Activation] 什麼時會觸發這個模式？
[Metadata]: {...}
```

## 使用場景

### 1. 錯誤記錄
記錄錯誤、根本原因、解決方案。

### 2. 經驗總結
總結成功或失敗的經驗。

### 3. 決策記錄
記錄重要決策的背景和理由。

## 優勢

- 結構化：便於檢索和分析
- 可執行：直接指導未來行為
- 可追蹤：記錄時間和來源
