# v0.6.113

This release continues the module-size cleanup for long-term maintainability.

## Changed

- Moved `vault automation ...` command handling and parser registration into
  `vault.cli_automation`.
- Kept `vault.cli.cmd_automation` as a small compatibility wrapper.
- Reduced `vault/cli.py` from 4413 lines to 3571 lines.
- Added `vault/cli_automation.py` at 869 lines, below the default new-module
  module-size gate.
- Lowered the `vault/cli.py` module-size baseline so it cannot silently grow
  back.

## Compatibility

The public automation CLI is unchanged. Existing commands, options, JSON output,
and human-readable output continue to behave the same.
