# v0.6.114

This release continues the module-size cleanup for long-term maintainability.

## Changed

- Moved Agent setup schedule, sync, and remote-reader template helpers into
  `vault.agent_setup_templates`.
- Kept `vault.agent_setup` compatibility imports for existing callers.
- Reduced `vault/agent_setup.py` from 4266 lines to 3408 lines.
- Added `vault/agent_setup_templates.py` at 899 lines, below the default
  module-size gate.
- Lowered the `vault/agent_setup.py` module-size baseline so it cannot silently
  grow back.

## Compatibility

The public `vault setup-agent` workflow is unchanged. Generated cron,
LaunchAgent, n8n, Supabase sync, remote-reader, and memory automation schedule
templates keep the same filenames and command behavior.
