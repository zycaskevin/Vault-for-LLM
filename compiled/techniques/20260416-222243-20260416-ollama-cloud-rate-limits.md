# Ollama Cloud限流問題

Ollama Cloud API 有並行模型數與用量限制，超限會導致請求失敗或延遲。使用時需注意請求模式與用量監控。

## 關鍵要點
- 並行模型數限制為3個，超限請求會被拒絕或延遲
- 用量以GPU時間計算，長提示與大模型會增加用量
- 過度使用可能觸發限流，需監控與調整請求模式
