# 傷疤格式更有用

## 核心洞察

記憶與行為改變直接關聯，不是日記式記錄。

## 經驗教訓

[Situation] 記錄記憶時
[Wrong] 記憶只是日記式記錄，沒有行為改變的意識
[Correct] 統一使用傷疤格式（Situation/Wrong/Correct/Behavior Change/Activation）
[Behavior Change] 所有記憶都使用傷疤格式，確保每條記憶都能改變行為
[Activation] 當記錄任何記憶時

## 傷疤格式

```
[Situation] 什麼情況？
[Wrong] 我做錯了什麼？
[Correct] 應該怎麼做？
[Behavior Change] 這會如何改變我未來的行為？
[Activation] 什麼時候會觸發這個模式？
```

## 價值

- 記憶系統性提升：記憶自動驗證行為改變
- 行為模式積累：長期來看會形成明確的行為模式
- 可執行性強：每條記憶都包含激活條件
