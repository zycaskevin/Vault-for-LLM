# Vault-for-LLM v0.6.119

This release continues the automation maintainability pass.

## What Changed

- Split automation cycle workspace and transcript-capture helpers into
  `vault.automation_cycle`.
- Kept `vault automation cycle` behavior unchanged.
- Reduced the size of `vault/automation.py` so the automation loop is easier to
  review and extend.

## Safety Notes

- Transcript capture is still opt-in.
- Captured transcript output remains candidate-only and content-hidden in cycle
  handoffs.
- Cycle workspace files are still written only under `reports/automation`.

## Upgrade

```bash
pip install -U "vault-for-llm[mcp]==0.6.119"
```
