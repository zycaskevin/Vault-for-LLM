# Flask 與 FastAPI 比較分析
## 比較維度
| 維度 | Flask | FastAPI |
|------|-------|---------|
| **類型** | 微型框架 | 現代 API 框架 |
| **學習曲線** | 簡單 | 中等 |
| **性能** | 良好 | 優秀（基於 Starlette） |
| **異步支援** | 有限 | 原生支援 |
| **文檔生成** | 需第三方 | 自動 OpenAPI 文檔 |
| **數據驗證** | 需擴展 | 內建 Pydantic |
