## 📊 三系統核心架構對比
```
┌─────────────────────────────────────────────────────────────────────────┐
│                         記憶系統架構對比                                  │
├─────────────────────┬───────────────────┬───────────────────────────────┤
│       系統           │      架構        │          核心特色             │
├─────────────────────┼───────────────────┼───────────────────────────────┤
│    MemPalace         │ 記憶宮殿層次      │ AAAK 壓縮、170 token 啟動    │
```

```
│      │   └── deploy（部署）                                 │
```
