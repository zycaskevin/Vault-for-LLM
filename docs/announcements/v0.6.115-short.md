# v0.6.115

This release continues the module-size cleanup for long-term maintainability.

## Changed

- Moved Agent setup startup, update-status, runtime adapter, and startup-doctor
  helpers into `vault.agent_setup_startup`.
- Kept `vault.agent_setup` compatibility imports for CLI and downstream callers.
- Reduced `vault/agent_setup.py` from 3408 lines to 2373 lines.
- Added `vault/agent_setup_startup.py` at 1071 lines, below the default
  module-size gate.
- Lowered the `vault/agent_setup.py` module-size baseline so it cannot silently
  grow back.

## Compatibility

The public `vault setup-agent`, `vault agent install-runtime-template`, and
`vault agent startup-doctor` workflows are unchanged. Generated MCP startup,
update-status, runtime adapter, and runtime playbook files keep the same names
and contracts.
