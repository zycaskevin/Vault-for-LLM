# 9000 端口衝突

## 錯誤信息
```
tcp        0      0 0.0.0.0:9000            0.0.0.0:*               LISTEN      1045/python3
```

## 根本原因
系統服務佔用 9000 端口，與應用配置衝突。

## 解決方案
1. 確認 9000 端口被系統服務佔用（PID 1045）
2. 禁止殺掉此進程（系統服務）
3. 修改應用配置，使用其他端口

## 預防措施
- 在 AGENTS.md 中明確約束：禁止使用 9000 端口
- 配置檢查腳本，自動檢測端口衝突
