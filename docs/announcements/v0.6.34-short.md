# Vault-for-LLM v0.6.34

This release connects the advanced Supabase read-policy template to MCP.

What's new:

- Added `vault_remote_search` for MCP remote profiles.
- The tool calls the Supabase `vault_search_readable` RPC generated by the advanced setup template.
- Hosted agents can now use `vault_remote_search` -> `vault_remote_map_show` -> `vault_remote_read_range`.
- Remote search passes `agent_id`, `include_private`, `max_sensitivity`, and `limit` to the RPC.
- Results return safe metadata and summaries only; raw full text stays out of the remote search payload.

Use this when Coze, n8n, or agents on another host need to search a Supabase-synced memory view without receiving the service role key.
