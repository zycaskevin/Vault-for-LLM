# Vault-for-LLM v0.7.19

v0.7.19 is a small consumer-onboarding polish release. It keeps the agent
tooling intact, but makes the setup output friendlier for people who only want
their agent to maintain the memory vault for them.

## Highlights

- Consumer `setup-agent` output now separates "For you" from "For your agent".
- Humans see the daily-memory workflow instead of a long engineering checklist.
- Agents and installers still get the full machine-readable setup payload with
  `--json`.
- The release keeps Python 3.10+ as the supported runtime floor.

## Install

```bash
pip install --upgrade "vault-for-llm[mcp]==0.7.19"
```
