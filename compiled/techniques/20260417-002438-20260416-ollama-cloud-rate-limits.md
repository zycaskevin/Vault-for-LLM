# Ollama Cloud限流問題

Ollama Cloud API 有並行模型數與用量限制，超限會導致請求失敗或延遲。使用時需注意請求模式與用量監控。

## 關鍵要點
- 同時最多3個雲端模型並行，用量以GPU時間計算
- 短時間連續呼叫或長prompt易觸發限流，可能導致timeout或429錯誤
- 使用本地模型或拆分長prompt，並監控用量避免超限
